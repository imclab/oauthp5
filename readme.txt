Compile the oauthP5 library using Ant.
From the Eclipse menu bar, choose Window > Show View > Ant. A tab with the title "Ant" will pop up on the right side of your Eclipse editor.
Drag the "resources/build.xml" file in there, and a new item "ProcessingLibs" will appear.
Press the "Play" button inside the "Ant" tab.
The library will be compiled to your Processing libraries folder.

Original Processing Library template instructions can be found at http://code.google.com/p/processing/wiki/LibraryTemplate
